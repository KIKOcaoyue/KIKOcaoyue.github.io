## 尺取法

**简单来说，尺取法就是~~像尺子一样~~两个指针一前一后(l,r)。r指针不断走，当不满足条件时，l指针也开始走，直到满足条件为止**

这大概就是尺取法的主要思想。

**我们直接来看本文的题目：**
以下是简单分析：
r移动的时候标记vis[a[r]]，当vis[a[r]]已经被标记过了，那么前面一段就不符合题意了，就要删掉，vis[a[l]]就要置成0，然后l移动。

举个例子：1 2 2 3 4

当r移动到第二个2的时候，显然就是不成立的，我们就要把最前面的1和2给删掉，然后从第二个2开始继续下一段的旅程。

代码在这里：

```c++
while(r<=n)
{
    if(!vis[a[r]])
    {
        vis[a[r]] = 1;
        r++;
        mx = max(mx,r-l);
    }
    else
    {
    	while(vis[a[r]])
    	{
            vis[a[l]] = 0;
            l++;
    	}
	}
}
mx = max(mx,r-l);
```

上面的就是整个尺取法的过程，相信大家在纸上写一写就能明白啦！

[poj3061](http://poj.org/problem?id=3061)

```c++
#include <cstdio>
#include <algorithm>
#define INF 0x3f3f3f3f
using namespace std;
const int maxn = 1e5+7;
int a[maxn];
int n,s;
int main()
{
    int T;
    scanf("%d",&T);
    while(T--)
    {   
        scanf("%d%d",&n,&s);
        int tot = 0;
        for(int i=1;i<=n;i++)
        {
            scanf("%d",a+i);
            tot += a[i];
        }
        if(tot<=s)
        {
            printf("0\n");
            continue;
        }
        int l = 1,r = 1;
        int mn = INF;
        int sum = 0;
        while(r<=n)
        {
            sum += a[r];
            r++;
            while(sum>s)
            {
                sum -= a[l];
                l++;
                mn = min(mn,r-l+1);
            }
        }
        printf("%d\n",mn);
    }
    return 0;
}
```

这道题目相当的简单，大概就是尺取法裸题，直接r先跑，当区间和大于s的时候再跑l，更新答案。



接下来的一道题是poj3320，题目较上一题有了些许难度，题目和牛客有点类似：

[poj3320](http://poj.org/problem?id=3320)

```c++
#include <iostream>
#include <cstdio>
#include <algorithm>
#include <set>
#include <map>
#define INF 0x3f3f3f3f
using namespace std;
const int maxn = 1e6+7;
int a[maxn];
int n;
set<int> s;
map<int,int> mp;
int main()
{
    scanf("%d",&n);
    for(int i=1;i<=n;i++)
    {
        scanf("%d",a+i);
        s.insert(a[i]);
    }
    int l = 1,r = 1;
    int mn = INF;
    int cnt = 0;
    while(r<=n)
    {
        if(!mp[a[r]])
        {
            cnt++;
            mp[a[r]]++;
        }
        else mp[a[r]]++;
        r++;
        while(cnt==s.size())
        {
            mn = min(mn,r-l);
            mp[a[l]]--;
            if(!mp[a[l]]) cnt--;
            l++;
        }
    }
    printf("%d\n",mn);
    return 0;
}
```

我们先让r跑到所有数字都出现过的点，然后再缩l，不断更新答案，直到不成立。



然后来一道~~硬核~~一点的题目

[poj2256](http://poj.org/problem?id=2566)

这道题目有正有负，乍一看根本没有单调性，没有单调性就根本没法尺取啊，怎么办呢？

我们可以先求前缀和，然后对前缀和排序，这样就有单调性了，大家想一想是不是这个道理。

然后我们对前缀和进行尺取，如果一段区间和tmp<t，那么区间就要变得更大，r++，如果tmp>t，那么区间需要变得更小,l++，不断比较最小值，不断更新需要的东西。



```c++
#include <cstdio>
#include <cstdlib>
#include <algorithm>
#define INF 0x3f3f3f3f
using namespace std;
typedef long long ll;
typedef pair<int,int> P;
const int maxn = 1e5+7;
ll a[maxn];
int n,k,t,q;
P pi[maxn];
bool cmp(P a,P b)
{
    return a.first<b.first;
}
int main()
{
    while(~scanf("%d%d",&n,&k))
    {
        if(n==0 && k==0) break;
        pi[0] = P(0,0);
        for(int i=1;i<=n;i++)
        {
            scanf("%d",a+i);
            pi[i].first = pi[i-1].first+a[i];
            pi[i].second = i;
        }
        sort(pi,pi+1+n,cmp);
        for(int i=1;i<=k;i++)
        {
            int mn = INF,mx = 0;
            scanf("%d",&q);
            int l = 1,r = 1;
            int mnn = INF;
            int ans = 0;
            while(r<=n)
            {
                int tmp = pi[r].first-pi[l-1].first;
                if(mnn>abs(tmp-q))
                {
                    mnn = abs(tmp-q);
                    mn = pi[l-1].second;
                    mx = pi[r].second;
                    ans = tmp;
                }
                if(tmp>q) l++;
                else if(tmp<q) r++;
                else break;
                if(l-1==r) r++;
            }
            if(mn>mx) swap(mn,mx);
            printf("%d %d %d\n",ans,mn+1,mx);
        }
    }
    return 0;
}
```



[poj2739](https://cn.vjudge.net/problem/POJ-2739)

```c++
#include <cstdio>
#include <iostream>
#include <algorithm>
#include <cstdlib>
#include <cstring>
using namespace std;
const int maxn = 1e5+7;
int pri[maxn],ispri[maxn];
int n;
void getprime()
{
    memset(ispri,1,sizeof(ispri));
    ispri[0] = ispri[1] = 0;
    for(int i=2;i<maxn;i++)
    {
        if(ispri[i]) pri[++pri[0]] = i;
        for(int j=1;j<=pri[0] && i*pri[j]<maxn;j++)
        {
            ispri[i*pri[j]] = 0;
            if(i%pri[j]==0) break;
        }
    }
}
int main()
{
    getprime();
    while(~scanf("%d",&n))
    {
        if(n==0) break;
        int cnt = 0;
        int l = 1,r = 1;
        int sum = 0;
        while(r<=n)
        {
            sum += pri[r];
            r++;
            while(sum>n)
            {
                sum -= pri[l];
                l++;
            }
            if(sum==n) cnt++;
        }
        printf("%d\n",cnt);
    }
    return 0;
}
```

这个题目就很简单了，一旦区间和大于n，我们就缩l，然后如果区间和等于n，我们就计数。



[poj2100](https://cn.vjudge.net/problem/POJ-2100)

```c++
#include <iostream>
#include <cstdio>
#include <vector>
#include <map>
using namespace std;
typedef long long ll;
typedef pair<ll,ll> P;
ll n;
vector<P> ans;
int main()
{
    while(~scanf("%lld",&n))
    {
        ll l = 1,r = 1;
        ll sum = 0;
        while(r*r<=n)
        {
            sum += r*r;
            r++;
            while(sum>n)
            {
                sum -= l*l;
                l++;
            }
            if(sum==n)
            {
                ans.push_back(P(l,r-1));
            }
        }
        printf("%d\n",ans.size());
        for(int i=0;i<ans.size();i++)
        {
            printf("%d ",ans[i].second-ans[i].first+1);
            for(int j=ans[i].first;j<=ans[i].second;j++) printf("%d ",j);
            puts("");
        }
    }
    return 0;
}
```

这个题目也很简单，基本上就是和区间和是一样的题目，注意要开long long ,不然就T掉了。

### 总结：

尺取法主要是高效的进行区间操作，可以在O(n)下对区间内进行统计，但是要注意的是，尺取法需要满足单调性，有的题目可以没有单调性，需要我们自己构造，这才是尺取法最难的地方。
